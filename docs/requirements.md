# 競売物件情報サイト 要件定義書

**作成日**: 2026-03-19
**参考サイト**: BIT 不動産競売物件情報サイト（https://www.bit.courts.go.jp）

---

## 1. プロジェクト概要

### 1.1 目的
裁判所の不動産競売物件情報を集約・表示し、ユーザーが条件に合った物件を効率的に検索・閲覧できるWebサイトを構築する。

### 1.2 ターゲットユーザー
- 不動産投資家
- 競売物件に興味のある一般ユーザー
- 不動産業者

---

## 2. フィルタリングルール

### 2.1 基本フィルタリング条件（ユーザー指定）

| 項目 | フィルタ条件 | 備考 |
|------|-------------|------|
| 建築年数 | 指定年以降の物件のみ表示 | 年数での絞り込み |
| 価格 | 価格帯での絞り込み | 売却基準価額ベース |
| 土地面積 | 指定面積以上の物件のみ表示 | 平米単位 |
| 自社掲載物件 | 除外する | 自社で掲載している物件は一覧から除外 |

### 2.2 デフォルトフィルタ（プリセット）

サイト初期表示時に以下の条件が自動適用される：

| 項目 | デフォルト値 | 備考 |
|------|-------------|------|
| 土地面積 | **90平米以上** | ユーザーが変更可能 |
| 建築年 | **1983年（昭和58年）以降** | 新耐震基準適用物件を対象とするため |

> **補足**: 1981年（昭和56年）に新耐震基準が施行されたが、建築確認から竣工までの期間を考慮し、1983年以降を基準とする。

## 3. 機能要件

### 3.1 画面一覧

| No | 画面名 | 概要 |
|----|--------|------|
| 1 | トップページ | 地域選択（地図ベース）、新着物件、物件総数表示 |
| 2 | 物件検索画面 | 各種条件による絞り込み検索 |
| 3 | 物件一覧画面 | 検索結果の一覧表示 |
| 4 | 物件詳細画面 | 個別物件の詳細情報表示 |
| 5 | 売却結果照会画面 | 過去の落札結果の閲覧 |
| 6 | 売却スケジュール画面 | 今後の売却予定の確認 |
| 7 | ヘルプ・用語集画面 | 使い方ガイド、FAQ、用語解説 |

### 3.2 トップページ

- 日本地図ベースの地域選択ナビゲーション（9ブロック）
  - 北海道、東北、北陸・甲信越、関東、東海、近畿、中国、四国、九州・沖縄
- 現在公開中の物件総数表示
- 新着物件情報の表示
- お知らせ欄

### 3.3 物件検索機能

#### 検索条件

| 条件 | 種別 | 詳細 |
|------|------|------|
| 地域 | セレクトボックス | 都道府県 → 市区町村（階層選択） |
| 裁判所 | セレクトボックス | 管轄地方裁判所・支部 |
| 物件種別 | チェックボックス | 戸建て / マンション / 土地 / その他 |
| 価格帯 | レンジ入力 | 売却基準価額の下限〜上限（万円単位） |
| 土地面積 | レンジ入力 | 下限〜上限（平米）、**デフォルト下限: 90平米** |
| 建築年 | セレクトボックス | 〇〇年以降、**デフォルト: 1983年以降** |
| 間取り | チェックボックス | 1R / 1K / 1LDK / 2LDK / 3LDK / 4LDK以上 |
| 築年数 | セレクトボックス | 〇年以内 |
| 入札状況 | ラジオボタン | 入札期間中 / 開札前 / すべて |
| 事件番号 | テキスト入力 | 直接指定検索 |

#### 除外条件

- 自社掲載物件は検索結果から自動除外
  - `properties`テーブルの`is_own_listing`フラグで管理
  - 管理画面から自社物件の登録・フラグ設定が可能

### 3.4 物件一覧画面

#### 表示項目

| 項目 | 説明 |
|------|------|
| 物件画像 | サムネイル表示（外観写真） |
| 物件種別 | 戸建て / マンション / 土地 等 |
| 所在地 | 都道府県〜番地 |
| 売却基準価額 | 評価書に基づく基準価格 |
| 買受可能価額 | 売却基準価額の80% |
| 土地面積 | 平米表示 |
| 建物面積 | 平米表示（建物ありの場合） |
| 建築年 | 西暦 / 和暦併記 |
| 間取り | LDK表記 |
| 管轄裁判所 | 地方裁判所名・支部名 |
| 入札期間 | 開始日〜終了日 |
| 開札期日 | 開札予定日 |

#### 表示オプション

- 一覧表示 / カード表示の切り替え
- ソート機能（価格順、面積順、新着順、開札日順）
- ページネーション（1ページあたり20件 / 50件 / 100件）

### 3.5 物件詳細画面

#### 基本情報セクション

- 物件画像（複数枚、スライダー表示）
- 物件種別
- 所在地（地図表示連携）
- 売却基準価額 / 買受可能価額
- 入札期間 / 開札期日

#### 物件スペックセクション

- 土地面積（平米 / 坪 併記）
- 建物面積（平米 / 坪 併記）
- 建築年（西暦 / 和暦 併記）
- 構造（木造、RC造 等）
- 間取り
- 階数
- 用途地域

#### 3点セットセクション

- 物件明細書（PDF閲覧・ダウンロード）
  - 不動産の表示、権利関係、占有状況、特記事項
- 現況調査報告書（PDF閲覧・ダウンロード）
  - 現況地目、建物種類・構造、占有者情報、現地写真
- 評価書（PDF閲覧・ダウンロード）
  - 周辺環境、評価額、算出根拠、図面・地図

#### その他セクション

- 交通アクセス（最寄り駅からの距離）
- 周辺施設情報
- 地図埋め込み（Google Maps等）
- お気に入り登録ボタン

### 3.6 売却結果照会機能

- 過去の落札結果の検索・閲覧
- 表示項目: 物件情報、売却基準価額、落札価額、落札倍率
- 過去3年分のデータを保持

### 3.7 売却スケジュール照会機能

- 裁判所別の今後の売却スケジュール一覧
- カレンダー表示対応
- 入札期間、開札期日の表示

## 4. 非機能要件

### 4.1 ユーザー機能

| 機能 | 詳細 |
|------|------|
| 会員登録 | メールアドレスによるユーザー登録 |
| ログイン | メール + パスワード認証 |
| お気に入り | 物件のブックマーク機能 |
| 検索条件保存 | よく使う検索条件の保存・呼び出し |
| 新着通知 | 保存した条件に合致する新着物件のメール通知 |

### 4.2 管理機能

| 機能 | 詳細 |
|------|------|
| 物件データ管理 | BITサイトからのデータ取得・更新 |
| 自社物件フラグ管理 | 自社掲載物件の登録・除外設定 |
| お知らせ管理 | サイトからのお知らせ投稿 |
| ユーザー管理 | 登録ユーザーの管理 |
| アクセス解析 | PV、検索条件の利用統計 |

### 4.3 パフォーマンス要件

- ページ読み込み時間: 3秒以内
- 検索レスポンス: 2秒以内
- 同時接続数: 100ユーザー以上
- 物件データ: 最大10,000件を想定

### 4.4 セキュリティ要件

- HTTPS通信
- SQLインジェクション対策
- XSS対策
- CSRF対策
- パスワードハッシュ化（bcrypt）

### 4.5 対応デバイス

- PC（Chrome, Firefox, Safari, Edge）
- タブレット（iPad対応）
- スマートフォン（iOS Safari, Android Chrome）
- レスポンシブデザイン対応

## 5. データモデル（主要エンティティ）

### 5.1 物件（properties）

```
- id: 物件ID（PK）
- court_id: 管轄裁判所ID（FK）
- case_number: 事件番号
- property_type: 物件種別（戸建て/マンション/土地/その他）
- address: 所在地
- prefecture: 都道府県
- city: 市区町村
- base_price: 売却基準価額
- minimum_bid_price: 買受可能価額
- land_area: 土地面積（平米）
- building_area: 建物面積（平米）
- construction_year: 建築年（西暦）
- structure: 構造
- floor_plan: 間取り
- floors: 階数
- zoning: 用途地域
- bid_start_date: 入札開始日
- bid_end_date: 入札終了日
- opening_date: 開札期日
- is_own_listing: 自社掲載フラグ（boolean）
- status: ステータス（入札中/開札前/売却済/取下げ）
- images: 物件画像パス（JSON）
- documents: 3点セットファイルパス（JSON）
- created_at: 登録日時
- updated_at: 更新日時
```

### 5.2 裁判所（courts）

```
- id: 裁判所ID（PK）
- name: 裁判所名
- region: 地域ブロック
- prefecture: 都道府県
- address: 住所
```

### 5.3 売却結果（sale_results）

```
- id: 結果ID（PK）
- property_id: 物件ID（FK）
- winning_bid: 落札価額
- bid_count: 入札件数
- result_date: 結果確定日
```

### 5.4 ユーザー（users）

```
- id: ユーザーID（PK）
- email: メールアドレス
- password_hash: パスワードハッシュ
- name: 氏名
- created_at: 登録日時
```

### 5.5 お気に入り（favorites）

```
- id: お気に入りID（PK）
- user_id: ユーザーID（FK）
- property_id: 物件ID（FK）
- created_at: 登録日時
```

### 5.6 自社物件除外リスト（excluded_listings）

```
- id: ID（PK）
- property_id: 物件ID（FK）
- reason: 除外理由
- registered_by: 登録者
- created_at: 登録日時
```

## 6. デフォルトフィルタ仕様

### 6.1 土地面積フィルタ

```
条件: land_area >= 90（平米）
適用タイミング: 検索画面初期表示時に自動適用
ユーザー変更: 可能（下限値を0に変更すれば全件表示）
```

### 6.2 建築年フィルタ

```
条件: construction_year >= 1983
適用タイミング: 検索画面初期表示時に自動適用
ユーザー変更: 可能
根拠:
  - 1981年（昭和56年）6月1日に新耐震基準施行
  - 建築確認〜竣工まで約1〜2年かかるため、
    1983年（昭和58年）以降竣工の物件を対象とすることで
    新耐震基準適用物件をほぼ確実にカバー
```

### 6.3 自社物件除外

```
条件: is_own_listing = false AND property_id NOT IN excluded_listings
適用タイミング: 常時（ユーザー変更不可）
管理: 管理画面から自社物件を登録
```
